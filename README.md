## Overview
This repository contains the implementation of a Natural Language Processing (NLP) model to perform sentiment analysis on customer reviews, completed as Task 2 during my Machine Learning Internship at CODTECH IT Solutions.

## Objective
To build a machine learning pipeline capable of understanding and classifying human text into positive or negative sentiments using mathematical vectorization techniques and predictive modeling.

## Tech Stack
* **Language:** Python
* **Libraries:** scikit-learn, pandas, matplotlib, seaborn
* **Techniques:** TF-IDF Vectorization, Logistic Regression
* **Environment:** VS Code (Jupyter Notebook)

## Deliverables
* **Text Preprocessing:** Converting raw string data into numerical features using `TfidfVectorizer` to highlight word importance.
* **Predictive Modeling:** Training a `LogisticRegression` algorithm to classify review sentiment.
* **Evaluation & Visualization:** Generating accuracy metrics, classification reports, and a heatmap-styled Confusion Matrix to assess model performance.
